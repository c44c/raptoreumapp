Help

This simple monitoring tool calculates the estimated income [excl. hosting] of your RTM Smartnode and displays it in a clear way. The data gets refreshed every minute automatically. To learn more about the Raptoreum project visit Raptoreum.com. Use the suffix #value=1800000 for example to change the value.

[P] toggle update bar [T] switch color theme [H] close help window
